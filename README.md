# 🎶 AI Lyric Generator (RNN - LSTM)

## 📌 Overview
This project generates song lyrics using a Recurrent Neural Network (LSTM).  
Users can enter a topic and generate lyrics with adjustable creativity.

---

## ⚙️ Features
- Generate lyrics from any topic
- Adjustable creativity using temperature
- Streamlit-based user interface
- Deep learning model using LSTM

---

## 🛠️ Tech Stack
- Python
- TensorFlow / Keras
- Streamlit

---

## ▶️ How to Run

### 1. Install dependencies
pip install tensorflow streamlit numpy

### 2. Train the model
python topic_lyric_generator_rnn.py

### 3. Run the app
streamlit run app.py
